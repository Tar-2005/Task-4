About Web Development Basics

Web development fundamentally consists of three pillars: HTML for structure, CSS for presentation, and JavaScript for interactivity. This project focuses on the first two to demonstrate that you can create professional, responsive websites with just HTML and CSS. Understanding these basics is crucial before moving to more complex frameworks and tools.

The "separation of concerns" principle is evident in this project: HTML handles content structure (home.html, about.html, contact.html), while CSS handles all visual styling (css/style.css). This approach makes maintenance easier and demonstrates professional coding practices.
