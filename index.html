Home Page

Welcome to our simple multi-page website built with pure HTML and CSS! This project demonstrates best practices for creating a clean, responsive, and professional website without using any frameworks or external libraries.

What You'll Find Here

This website showcases fundamental web development skills including:

  • Semantic HTML5 structure
  • Responsive CSS with Flexbox layout
  • Consistent navigation across multiple pages
  • Active link highlighting for better user experience
  • Clean and organized file structure
  • Mobile-first responsive design

Project Structure

The project is organized into a clear folder hierarchy: 

project-folder/
├── index.html (You are here)
├── about.html
├── contact.html
├── css/
│   └── style.css (Single stylesheet for all pages)
└── assets/ (For future images and media)

Getting Started

To explore the website, use the navigation links at the top. Each page uses the same CSS file to maintain a consistent design, and navigation links dynamically highlight the current page using the .active class.
